# packages-playground
> :wave: :earth_americas: a playground for **Packages** :roller_coaster:

## Build
```shell
docker build -t ghcr.io/swinton/echo:$VERSION .
```

## Push
```shell
docker push ghcr.io/swinton/echo:$VERSION
```

## Run
```shell
docker run --rm ghcr.io/swinton/echo:$VERSION hello world
```
